# P10M2 Core Finalization Before LoRA

P10M2 adds standardized knowledge chunks, offline hybrid retrieval, evidence citations, RAG guard checks, report export, deterministic safety redteam, RAG eval, Final Eval v2, Docker local run defaults, anonymous failure memory, and the LoRA integration contract.

## Safety Position
The assistant remains a structured inquiry and risk reminder system. It does not diagnose, prescribe, or replace clinicians. RAG and future LoRA extraction cannot overwrite deterministic risk state.
